<CR> then "~" then "?" : SSH client key-binding help.
<CR> then "~" then "." : SSH client key-binding help.
<CR> then "~" then <CTRL>+"Z" : Suspend client SSH process.
<CR> then "~" then "?" : SSH client key-binding help.
<CR> then "~" then "~" : Send literal "~" to remote session.
